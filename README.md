# Cybersecurity Intrusion Detection using SVM

## Overview

This project uses Support Vector Machine (Linear SVM) to classify network traffic into different attack categories.

## Features

- Data preprocessing
- Feature scaling
- Linear SVM
- FastAPI deployment
- Interactive Swagger UI

## Technologies

- Python
- Scikit-learn
- FastAPI
- Joblib
- NumPy
- Pandas

## Run in Command Prompt

python train_cybersecurity_IDS.py

python -m uvicorn app_cybersecurity_IDS:app --reload
